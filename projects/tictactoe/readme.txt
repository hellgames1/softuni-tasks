Tic-Tac-Toe against a bot

Basically you can play Tic-Tac-Toe against a decently working bot. It tries to think one move ahead. It has three types of moves

random - when it doesn't matter where it will put an O
offensive - when it realizes it can win the next round if it puts an O somewhere
defensive - when it realizes you might win the next round if it doesn't put an O somewhere

There is a 10% chance it won't realize those things (in order to make it more human-like)

The .exe file is just the python code packaged with the necessary libraries so anyone can run it

The .en.py file is an English version of the program, since the original is in Bulgarian. The comments have also been translated.

============================================================================================================

Морски шах срещу бот

Можеш да играеш морски шах срещу доста добре работещ бот. Опитва се да мисли с едно движение напред. Има три вида движения

произволно - когато няма значение къде ще сложи О
атакуващо - когато се усети че може да спечели следващия рунд с определено движение
защитно - когато се усети че може ти да спечелиш следващия рунд ако не направи определено движение

Има 10% шанс да не се "усети", за да е повече като човек

.exe файлът е просто Python кода пакетиран с нужните библиотеки за да може директно да се пусне от всеки

.en.py файлът е английски превод на програмата, тей като оригиналът е на български. Коментарите също са преведени.
